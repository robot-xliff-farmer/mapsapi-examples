Searching for an object by name is implemented using the function. However, this function searches among the objects placed on the regular or public map.

What if you want to organize the search across custom objects and integrate it with the Yandex.Maps API? Write your own search provider, which can be used in the function and the "Search on map" control.

The search provider must implement the interface. This example shows how to write your own provider and connect it to the standard "Search on map" control.
