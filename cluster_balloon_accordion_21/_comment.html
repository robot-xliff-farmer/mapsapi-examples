The example uses the "cluster#balloonAccordion" cluster balloon layout.
This layout displays a list of geo objects, and clicking on an item shows information about the geo object.
The information about the selected geo object is set by a separate layout and can be configured using 'clusterBalloonItemContentLayout'.
This example sets a custom layout with information about a geo object.

Layouts of objects can be created using the factory , using text templates.
Text templates generate HTML content for the layout based on a hash with data, to pass to the layout constructor.
In this example, the data provider is the geo object.
