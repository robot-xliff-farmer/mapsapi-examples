The clusterer is used for combining placemarks that are near each other in a group (cluster).

Clusterization is implemented by the class. As parameters to the constructor, you can pass clusterer options: the style for displaying the cluster icon, the size of its cell, and so on. For the list of available options, see the relevant section of the reference guide.

To add geo objects to the clusterer, use the add method. Either a separate geo object or an array of geo objects can be passed as the parameter.

The example shows how to change the color of the icon or cluster on mouse hover.
