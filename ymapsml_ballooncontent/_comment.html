Templates allow you to make custom HTML representations of geo objects, balloons, and hints.

To define a template, use therepr:Representation element, which contains the HTML code (the template body) in the repr:text container.

When using a template, access to the value of anyXML element located inside the ymaps:GeoObject or ymaps:GeoObjectCollection element requires accessing the corresponding substitution variable.

To define a substitution, use a string in the format $[substitution_name]|default_value]. The substitution name is the location of this XML element inside the element that the style is applied to. The location is defined using an ordered list of the names of all parent elements and the name of the element itself. A dot is used as the separator.

For more information about templates, see the documentation.
