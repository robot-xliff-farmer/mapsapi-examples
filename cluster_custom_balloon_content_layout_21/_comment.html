This example creates a custom layout for the cluster balloon content.
The layout is defined by the 'clusterBalloonContentLayout' option.
Layouts of objects can be created using the factory, using text templates.
Text templates form the layout's HTML content based on a hash with the data passed to the layout constructor.
In this example, the data provider is the cluster that the balloon was opened on.

The layout lists and outputs the 'placemarkId' and 'balloonContentHeader' fields of all the cluster's geo objects using the 'for' construction.
