The Yandex.Maps Api consists of a large number of interdependent modules.
By default, enabling the API loads a standard set of modules (package.full), which includes everything necessary for working with the API.
A developer who is using the Yandex.Maps API can create custom modules.

This example creates a custom control for a custom layout.
Both the control and the layout are created as separate modules.
To define a custom module, use the define method.
After the map is created, use the require method to request to get the custom module.

For more information about how modules work, see the section Modules.
