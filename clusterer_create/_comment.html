The clusterer is used for combining closely positioned placemarks in a group (cluster).

The clusterer is implemented by the class. You can pass clusterer options to the constructor as parameters: the display style for the cluster icon, the cell size, and so on. For the list of available options, see the relevant section of the Reference Guide.

To add geo objects to the clusterer, use the add method. Either a separate geo object or an array of geo objects can be passed as the parameter.
